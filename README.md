# AI Resume Reviewer

An AI-powered resume analyzer that gives actionable feedback 
against any job description.

## Live Demo
https://ai-resume-checker-blond.vercel.app/

## Features
- Paste resume text or upload PDF
- AI analyzes against job description
- ATS match score
- Strengths, weaknesses, missing keywords
- Actionable improvements

## Built With
- React + TypeScript
- Tailwind CSS
- Zustand (state management)
- Cohere AI API
- PDF.js (PDF parsing)
- Vercel (deployment)
